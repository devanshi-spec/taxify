# Release Notes

This folder contains all release notes for the US Tax Form Extractor project.

## Release History

| Version | Date | Description |
|---------|------|-------------|
| v1.3.1 | 16th Jan 2026 | Critical bug fix for W-2 Box 12 shifting issue |
| v1.3.0 | 16th Jan 2026 | Major architectural split of API and new Registry system |
| v1.2.0 | 12th Jan 2025 | Added support for Form K-1, 8804, and 8805 |
| v1.1.0 | 26th Dec 2024 | Bug fixes for W-2 form extraction (name+address, state wages/tax) |
| v1.0.0 | 22nd Dec 2024 | Initial release with W-2 and 1099-INT support |

## Naming Convention

Release note files follow this naming pattern:
- `RELEASE_v{major}.{minor}.{patch}.md` - Main release notes
- `RELEASE_v{major}.{minor}.{patch}_R{revision}.md` - Revision releases
